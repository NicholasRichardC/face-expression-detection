# Face Expression Detection (Streamlit)

Aplikasi deteksi ekspresi wajah real-time menggunakan:
- TensorFlow / Keras
- OpenCV (headless)
- Streamlit

Ekspresi yang dideteksi:
- Angry
- Fear
- Happy
- Neutral
- Sad
- Surprise

Cara deploy di Streamlit Cloud:
1. Upload project ini ke GitHub
2. Tambahkan model keras (model.keras)
3. Deploy ke https://share.streamlit.io
